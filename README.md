# RentConnect
A modern, fully responsive rental property marketplace built with pure HTML, CSS &amp; JavaScript  no frameworks, no backend.
🏠 RentConnect — Uganda's Rental Marketplace

RentConnect is a fully responsive, production-quality rental property 
marketplace built entirely with vanilla HTML5, CSS3, and JavaScript. 
No frameworks. No backend. No database. Just clean, fast, modern web code.

Built as a complete multi-page website with 9 pages, 6 CSS files, 
6 JS modules, and a local JSON data layer — ready to open in a browser 
and run instantly.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📄 PAGES
  • Home          — Hero, search, featured listings, stats, testimonials
  • Properties    — Filterable grid with sidebar, sort, and pagination
  • Property Detail — Image gallery, amenities, booking form, similar listings
  • Landlords     — Landing page with pricing plans and FAQ
  • Dashboard     — Landlord portal with stats, property table, add/edit forms
  • Login         — Authenticated login with validation
  • Register      — Renter & landlord registration with password strength meter
  • About         — Company story, mission, team section
  • Contact       — Contact form + FAQ accordion

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✨ FEATURES
  • Glassmorphism UI with smooth gradients and soft shadows
  • Dark mode toggle (persisted via localStorage)
  • Scroll-triggered reveal animations
  • Hero typewriter effect
  • Animated counters
  • Property search, filter, and sort (all client-side)
  • Save favourites with localStorage
  • Form validation with real-time feedback
  • Password strength indicator
  • Toast notification system
  • Scroll progress bar
  • Fully responsive — mobile, tablet, desktop
  • Landlord dashboard with add / edit / delete property
  • FAQ accordion
  • Page loader animation

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🛠 TECH STACK
  • HTML5 (semantic, SEO-friendly structure)
  • CSS3 (custom properties, grid, flexbox, animations)
  • Vanilla JavaScript (ES6+, modular, no dependencies)
  • Google Fonts — Outfit + Inter
  • Unsplash for sample property images
  • localStorage for auth state, favourites, and dashboard data

🚀 GETTING STARTED
  1. Clone or download the repository
  2. Open index.html in any modern browser
  3. No installation, no build step, no server required
